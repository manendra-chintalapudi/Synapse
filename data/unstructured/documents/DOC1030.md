# RCA Report — DOC1030
**Document type:** Root Cause Analysis Report
**Related failure:** F1121  |  **RCA record:** RCA1121
**Equipment:** EQ-BF-03 — Blast Furnace Line 3 (Blast Furnace)
**Location:** Pune Unit - Bay 2
**Event timestamp:** 2025-10-25 16:01:45
**Failure mode:** Tool Wear Failure (source flag: TWF)

## 1. Incident Summary
On 2025-10-25 16:01:45, EQ-BF-03 (Blast Furnace Line 3) experienced a tool wear failure that
interrupted normal operation. The condition-monitoring system captured the following process values at the time of
the event: air temperature 296.3 K, process temperature 306.4 K,
rotational speed 2865 rpm, torque 18.7 Nm, and accumulated tool wear
224 minutes. The event was logged by the equipment engineer on shift and escalated for formal
root cause analysis under the plant's CMMS reliability programme.

## 2. Investigation & Findings
A cross-functional review examined operating logs, the sensor trace immediately preceding the trip, recent maintenance
history and the applicable operating procedure for the blast furnace. Trends in the minutes before the event were
compared against the equipment's qualified operating envelope. The investigation focused on whether the observed values
represented a developing fault or a transient excursion, and whether any preventive-maintenance trigger had been missed.

## 3. Root Cause
Root cause was excessive tool wear: accumulated tool-wear minutes exceeded the qualified service life, degrading process capability and surface quality before scheduled replacement. Extended run time between changes was the main driver.

## 4. Corrective Action
Replaced the worn tool and implemented tool-wear-based (rather than purely time-based) replacement scheduling.

## 5. Preventive Recommendations
To reduce recurrence risk, the team recommends reinforcing the condition-based maintenance thresholds associated with
this failure mode, updating the relevant operating procedure, and adding the event to the equipment's reliability
history for trend analysis. Spare-part availability for the affected sub-assembly should be confirmed, and operators
briefed on the early-warning indicators identified during this investigation.
